📱 Installer Beyma Flow

Android Chrome / Edge
1. Ouvre Beyma Flow dans Chrome
2. Clique les 3 points ⋮ en haut à droite
3. Clique "Ajouter à l'écran d'accueil"
4. Confirme → l'icône apparaît sur ton bureau ✅
Xiaomi / Redmi MIUI
⚠️ Chrome sur MIUI peut bloquer l'installation

Solution 1 : Utilise Firefox → menu ⋮ → "Installer"
Solution 2 : Paramètres Chrome → "Autoriser les applications"
Solution 3 : Mi Browser → menu → "Ajouter à l'écran"
iPhone Safari
⚠️ Sur iPhone, utilise Safari uniquement

1. Ouvre Beyma Flow dans Safari
2. Clique l'icône Partager ↑ en bas
3. Défile et clique "Sur l'écran d'accueil"
4. Confirme → icône sur le bureau ✅
