MyEcho
Connectez votre Supabase
MyEcho utilise Supabase pour stocker vos fichiers audio dans le cloud. Créez un projet gratuit en 2 minutes et collez vos clés ci-dessous.
1

Créer un projet Supabase

Allez sur supabase.com → New Project → choisissez un nom et un mot de passe de base de données.

2

Récupérer vos clés

Dans votre projet → Settings → API. Copiez l'URL du projet et la clé anon/public.

3

Créer le storage bucket

Storage → New Bucket → nommez-le myecho-audio → cochez Public.

4

Créer les tables SQL

SQL Editor → collez et exécutez le script fourni dans schema.sql ci-joint.

⚠️ Fichier ouvert en local
Pour l'inscription/connexion, Supabase nécessite une URL autorisée.
Allez dans Supabase → Authentication → URL Configuration → ajoutez http://localhost dans Redirect URLs.

Ou hébergez le fichier sur Netlify pour éviter ce problème.

Vos clés sont stockées localement dans votre navigateur.
Elles ne sont jamais envoyées à des tiers.
